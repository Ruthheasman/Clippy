Clippy Web Widget

The AI cursor companion, running in your browser. No install required.

BYOK — Bring Your Own Key

This page has the Clippy web widget injected. You should see a floating chat panel (or a blue 📎 button in the bottom-right corner). Click it to start chatting.

Three Ways to Use It

1. Script Tag

Add to any HTML page:

<script src="inject.js"></script>

2. Bookmarklet

Drag this to your bookmark bar, then click it on any page:

📎 Clippy

3. Browser Console

Paste into DevTools console:

fetch('inject.js').then(r=>r.text()).then(eval)

Programmatic Control

After injection, window.ClippyWidget is available:

ClippyWidget.show()     // Open the panel
ClippyWidget.hide()     // Close the panel
ClippyWidget.toggle()   // Toggle visibility
ClippyWidget.send(text) // Send a message programmatically
ClippyWidget.destroy()  // Remove the widget entirely

Supported Providers

Claude (Anthropic), OpenAI, Gemini (Google), and Ollama (local). Just paste your API key and go.
